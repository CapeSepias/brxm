Supported stacks

Hippo CMS requires a stack of system software to be able to run. Hippo CMS support is only OS type dependent, not stack dependent. I.e.: given a particular OS type, you can choose any piece of system software from the sets given below. Apart from you choice of OS type, none of your choices reduces your others sets of options.

  • Tomcat runs slow on Microsoft OS'ses so is preferably run on Linux. The cause is that Tomcat makes excessive use of thread switching, which eventually has to be taken care of by the OS and MS OS's are not very good at that.
