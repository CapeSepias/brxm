Plugin examples

This is a Plugin Panel. A Plugin Panel is a Panel that implements the method "void update(AjaxRequestTarget target, JcrNodeModel model)". Through that interface it receives update events from other plugins (like the tree or the menu) when it needs to refresh it's content with a new Model.

This example plugin could grow into a plugin central.

Context aware dialog

A 'context-aware' dialog is a dialog whose implementation (java+markup) can be determined at runtime. To see this in action: select a(ny) node, add a Property with name 'renderer' and as value the classname of a(ny) subclass of AbstractDialog (like org.hippocms.repository.plugins.admin.menu.move.MoveDialog). Select the Node again and click on this link.
